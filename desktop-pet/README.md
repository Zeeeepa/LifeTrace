# LifeTrace 桌宠

一个基于 Electron 的桌面日程管理应用，以桌宠的形式呈现。

## 技术栈

- **Electron**: 桌面应用框架
- **React 19**: UI 框架
- **TypeScript**: 类型安全
- **Vite**: 构建工具

## 开发

### 安装依赖

```bash
npm install
# 或
pnpm install
```

### 运行开发环境

```bash
npm run dev
```

这会同时启动 Vite 开发服务器（端口 5174）和 Electron 应用。

### 构建

```bash
npm run build
```

## 窗口配置

- 窗口大小: 300x400 像素
- 无边框、透明背景
- 始终置顶
- 不显示在任务栏
- 默认位置: 屏幕右下角

## 端口配置

- 开发服务器端口: **5174**（与 frontend 的 3000 端口区分）
